## Descripción del Servicio

### Qué Es QuizLab

QuizLab es una plataforma en línea que:

✅ Ofrece tests interactivos (personalidad, trivia, curiosidades, etc.)  
✅ Muestra resultados personalizados  
✅ Permite compartir resultados en redes sociales  
✅ Vende e-books, cursos, y contenido digital  
✅ Muestra anuncios (CPM/CPC)  
✅ Integra enlaces de afiliados (Hotmart, etc.)  
✅ Recopila análisis de comportamiento  

### Lo Que NO Somos

❌ Asesor médico/psicológico (los tests son solo entretenimiento)  
❌ Servicio financiero (no manejamos inversiones)  
❌ Plataforma de redes sociales (aunque puedas compartir)  
❌ Proveedor de educación formal (no expedimos certificados válidos)  

### Disponibilidad

Intentamos mantener QuizLab disponible 24/7, pero:

- ⚠️ Sin garantía de 99.9% uptime
- ⚠️ Mantenimiento programado puede causar downtime
- ⚠️ Factores externos (ataques, desastres naturales) pueden afectar
- ⚠️ Buscamos notificarte con anticipación

### USO PERMITIDO ✅

```

✅ Compartir resultados públicamente

```

### USO PROHIBIDO ❌

**Actividades Prohibidas:**

```
❌ COMPORTAMIENTO ILEGAL
  ├─ Violar leyes locales/nacionales/internacionales
  ├─ Fraude, estafa, suplantación
  ├─ Venta de acceso (tu cuenta no es transferible)
  └─ Lavado de dinero, financiamiento de terrorismo

❌ ACCESO NO AUTORIZADO
  ├─ Hackear sistemas de QuizLab
  ├─ Intentos de acceso sin permiso
  ├─ Reverse engineering
  ├─ SQL injection, XSS, DDoS
  └─ Escaneo de vulnerabilidades sin permiso

❌ ABUSO DE CONTENIDO
  ├─ Spam de mensajes/comentarios
  ├─ Contenido sexual/violento/odio
  ├─ Acoso a otros usuarios
  ├─ Publicidad no autorizada
  ├─ Malware/phishing
  └─ Impersonación

❌ ABUSO COMERCIAL
  ├─ Scrapar/descargar masivamente contenido
  ├─ Revender nuestro contenido
  ├─ Usar para competencia directa
  ├─ Vender acceso a otros usuarios
  └─ Bot farming (crear múltiples cuentas)

❌ MANIPULACIÓN DE DATOS
  ├─ Falsificar resultados de tests
  ├─ Intentar ver respuestas de otros
  ├─ Alterar tus datos de compra/pago
  └─ Manipular algoritmos de recomendación

❌ EVASIÓN DE RESTRICCIONES
  ├─ VPN para evadir geobloqueos
  ├─ Usar cuentas secundarias cuando estés baneado
  ├─ Evadir moderación o controles parentales
  └─ Múltiples cuentas para múltiples reembolsos
```

### Consecuencias del Incumplimiento

**Nivel 1: Advertencia**
- Primera violación menor
- Te notificamos por email
- Describes qué pasó

**Nivel 2: Suspensión (Temporal)**
- Violación reiterada
- Cuenta deshabilitada por 7-30 días
- Pérdida de acceso temporalmente

**Nivel 3: Terminación (Permanente)**
- Violaciones graves (fraude, hacking, abuso)
- Cuenta eliminada para siempre
- Datos eliminados (excepto registros legales)
- Prohibición de crear nuevas cuentas
- IP baneada

**Escalada a Autoridades**
- Fraude > $100
- Hacking/acceso no autorizado
- Contenido ilegal
- Reportamos a:
  - 🇪🇸 Policía Nacional (España)
  - 🇨🇴 DIJÍN (Colombia)
  - 🇺🇸 FBI (USA, si aplica)

```
support@quizlab.com

```

```
escalation@quizlab.com

Cuerpo: Tu queja anterior + por qué no estás satisfecho
```

```
Si aún no está resuelto:

Arbitrador neutral (país aplicable):

COSTO: Gratuito para ti (QuizLab paga)
```
