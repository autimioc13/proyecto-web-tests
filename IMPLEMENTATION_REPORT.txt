KEY FEATURES
================================================================================

User Sidebar (Unchanged):
  - 96px fixed width, icon-only navigation
  - 4 items: Home, Tests, Dashboard, Profile
  - Sound and Theme toggles
  - Glasmorphic design
  - Dark mode support
  - Active state highlighting

Admin Sidebar (New):
  - 250px fixed width
  - 5 organized menu categories
  - 20 total menu items
  - Collapsible category groups
  - Active state with left border
  - Premium plan footer
  - Mobile hamburger menu
  - Glasmorphic design
  - Dark mode support

Role Detection:
  - Multi-layer fallback system
  - Layer 1: admin_session cookie
  - Layer 2: ADMIN_EMAILS environment variable
  - Layer 3: Supabase user.role (future)
  - Layer 4: Default to regular user

Route Protection:
  - Middleware validates /admin/* routes
  - Server-side role check in AdminLayout
  - Non-admins redirected to home
  - Expired sessions redirect to login
  - Cookie-based session (24-hour validity)
