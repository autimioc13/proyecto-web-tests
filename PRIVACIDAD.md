## Introducción

QuizLab ("**nosotros**", "**nuestro**" o "**la Plataforma**") respeta tu privacidad. Esta Política de Privacidad explica:

- **QUÉ** información recolectamos
- **CÓMO** la usamos
- **CON QUIÉN** la compartimos
- **TUS DERECHOS** sobre tu información

**Aplicable a**: Usuarios de www.quizlab.com y aplicaciones relacionadas, operando bajo leyes globales (GDPR, CCPA, Habeas Data, etc.).

## Información que Recolectamos

### 1. Información de Registro y Cuenta

| Datos | Por qué | Base Legal |
|-------|---------|-----------|
| Email | Crear cuenta, autenticación | Contrato |
| Nombre/Pseudónimo | Perfil público | Consentimiento |
| Contraseña (hash) | Seguridad | Contrato |
| Foto de perfil (opcional) | Personalización | Consentimiento |

### 2. Información de Comportamiento

**Mientras usas la plataforma, registramos:**

```
┌─ TESTS/QUIZZES
├─ Qué tests iniciaste
├─ Preguntas en las que hiciste clic
├─ Tiempo en cada pregunta
├─ Tus respuestas (ANÓNIMAS por defecto)
├─ Resultado final
└─ Si compartiste el resultado

┌─ NAVEGACIÓN
├─ Páginas visitadas
├─ Tiempo en cada página
├─ Botones que presionaste
├─ Búsquedas realizadas
└─ Filtros aplicados

┌─ COMERCIAL
├─ Tests/productos comprados
├─ Precios pagados
├─ Método de pago (solo tipo: tarjeta/transferencia)
├─ E-books descargados
├─ Afiliados visitados
└─ Newsletter subscritos
```

**Motivo**: Analítica, personalización, mejora de experiencia, detección de fraude.

### 3. Información de Dispositivo

```
IP Address          → Ubicación general, detección de fraude
User-Agent          → Tipo de dispositivo (móvil/desktop)
Cookie/Local Storage → Preferencias, sesión, identificador anónimo
Timestamps          → Cuándo accediste
Identificador único → Rastrear sesiones
```

### 4. Información de Terceros

- **Pagos**: Tu procesador de pagos (Stripe, etc.) nos comparte estado de transacción
- **Redes sociales**: Si inicias sesión con Google/Facebook, nos comparten tu email + info pública
- **Afiliados**: Si vienes de un link de afiliado (Hotmart), rastreamos esa referencia

### 5. Información Sensible (Explícitamente Recolectada)

- **Datos de personalidad**: Si haces tests de personalidad/psicológicos, tus respuestas revelan información personal
- **Intereses**: Qué tópicos exploras (viajes, relaciones, carrera, etc.)
- **Comportamiento**: Con qué frecuencia usas la plataforma
- **Ubicación aproximada**: Por tu IP

**Estos datos son tratados con máxima confidencialidad.**

## Cómo Usamos tu Información

### 1. Operación de la Plataforma (ESENCIAL)

✅ Procesar tus pagos  
✅ Entregar tests y e-books  
✅ Autenticarte y mantener tu sesión  
✅ Responder tus mensajes/soporte

✅ Ajustar dificultad según tu desempeño  
✅ Mejorar la plataforma (qué funciona, qué no)  
✅ A/B testing de nuevas features  

**Base legal**: Interés legítimo (Art. 6.1.f GDPR) / Consentimiento (CCPA)

### 3. Marketing y Comunicación (CON CONSENTIMIENTO)

⚠️ **Solo si lo autorizaste:**

✅ Enviarte emails sobre nuevos tests/promociones  
✅ Notificaciones push (en app)  
✅ Mensajes SMS (solo si diste permiso)  
✅ Ofertas personalizadas basadas en tus intereses  

**Puedes desuscribirse en cualquier momento** → [Gestionar Preferencias](#)

**Base legal**: Consentimiento (Art. 7 GDPR, CCPA §1798.100)

### 4. Monetización y Análisis (ANÓNIMIZADO)

✅ Mostrar anuncios relevantes a tu perfil (CPM/CPC)  
✅ Analizar qué tests generan ingresos  
✅ Optimizar ads por región/dispositivo  
✅ Reportar métricas a anunciantes (datos agregados, sin identificarte)  

**Base legal**: Interés legítimo (Art. 6.1.f GDPR)

- Qué tests jugaste (nombre)
- Cuántos usuarios tienen tal resultado
- Tasas de completación por región
- **PERO NO**: Tu email, respuestas específicas, nombre real

### 5. Cumplimiento Legal

✅ Evitar fraude y abuso  
✅ Reportar actividades ilegales a autoridades  
✅ Defender nuestros derechos legales  
✅ Cumplir órdenes judiciales  

**Base legal**: Obligación legal (Art. 6.1.c GDPR)

### 6. Análisis de Comportamiento (Behavioral Profiling)

**Creamos perfiles de usuario para:**

```
Personalización de contenido
Detección de fraude
Recomendaciones
Optimización de precios (variable pricing)
Segmentación de marketing
```

## Almacenamiento y Seguridad

### 1. Dónde Guardamos tus Datos

```
PRODUCCIÓN (USA)
├─ AWS RDS (Base de datos encriptada)
├─ S3 (E-books, imágenes)
└─ CloudFront (CDN caché)

BACKUP (Brasil)
├─ AWS S3 Cross-Region Replication
└─ Automático cada 24 horas

ARCHIVO (Colombia)
└─ Encriptado, acceso limitado (legal hold)
```

### 2. Medidas de Seguridad

| Capa | Protección |
|-----|-----------|
| **Transporte** | TLS 1.3 (HTTPS obligatorio) |
| **Base de datos** | Encriptación AES-256 en reposo |
| **Passwords** | Hash bcrypt + salt único |
| **APIs** | JWT + rate limiting |
| **Acceso** | 2FA obligatorio para admin |
| **Auditoría** | Logs de acceso encriptados |
| **Monitoreo** | Detección automática de anomalías |

### 3. Incidentes de Seguridad

Si tus datos se ven comprometidos:

1. ⏱️ **Notificación en 72 horas** (GDPR deadline)
2. 📧 Correo explicando qué pasó
3. 🛡️ Oferta de monitoreo de crédito (si aplica)
4. 📋 Reporta a autoridades (GDPR Art. 33)

## Tus Derechos

### En la UE/EEE (GDPR)

Tienes derecho a:

#### 1. **Derecho de Acceso** (Art. 15)
- Solicita una copia de TODOS tus datos
- Formato: ZIP descargable
- Plazo: 30 días
- **Cómo**: [Descargar mis datos](#)

#### 2. **Derecho de Rectificación** (Art. 16)
- Corregir datos inexactos
- Completar datos incompletos
- **Cómo**: [Editar perfil](#)

#### 3. **Derecho al Olvido** (Art. 17) - "Derecho a ser Olvidado"
- Solicitar eliminación completa
- Excepciones: obligaciones legales, fines históricos
- Plazo: 30 días
- **Cómo**: [Solicitar eliminación](#)

#### 4. **Derecho de Restricción** (Art. 18)
- Limitar cómo usamos tus datos
- Ej: "Solo para contrato, sin marketing"
- **Cómo**: [Gestionar preferencias](#)

#### 5. **Derecho de Portabilidad** (Art. 20)
- Obtener tus datos en formato transportable
- Enviarlos a otra plataforma
- **Cómo**: [Exportar datos](#)

#### 6. **Derecho de Objeción** (Art. 21)
- Optar por NO participar en:
  - Perfilado (behavioral profiling)
  - Marketing directo
  - Análisis de datos
- **Cómo**: [Optar por no participar](#)

#### 7. **Derechos Relativos a Decisiones Automatizadas** (Art. 22)
- Derecho a NO ser sujeto de decisiones automatizadas
- Ej: "No quiero que un algoritmo decida si me muestran ads"
- **Cómo**: [Solicitar revisión manual](#)

### En California (CCPA)

Tienes derecho a:

| Derecho | Descripción | Plazo |
|--------|-----------|--------|
| **Know** | Saber qué datos tenemos | 45 días |
| **Delete** | Pedir eliminación | 45 días |
| **Opt-Out** | No vender datos | Inmediato |
| **Correct** | Corregir datos | 45 días |
| **No Discrimination** | No te castigamos por ejercer derechos | - |

### En Colombia (Habeas Data)

Tienes derecho a:

```
✅ Acceso a tus datos
✅ Rectificación si son incorrectos
✅ Cancelación (eliminación)
✅ Oposición a ciertos procesamiento
✅ Revocar consentimiento
✅ No ser objeto de decisiones automatizadas
```

## Retención de Datos

### Cuánto Tiempo Guardamos Tus Datos

```
CUENTA ACTIVA
├─ Datos de registro    → Mientras tengas cuenta
├─ Historial de tests   → Mientras tengas cuenta
├─ Compras/pagos        → 7 años (legal: fisco)
└─ Comportamiento       → 2 años (después anonimizado)

CUENTA ELIMINADA
├─ Datos personales     → Eliminados inmediatamente
├─ Historial anónimizado → 3 años (análisis estadístico)
├─ Transacciones        → 7 años (requisito legal)
└─ Logs de seguridad    → 1 año (auditoría)

NUNCA GUARDAMOS
├─ Passwords en texto plano
├─ Números de tarjeta completos (Stripe guarda, no nosotros)
├─ Datos después de SOLICITAR ELIMINACIÓN
└─ Información que no es estrictamente necesaria
```

**Excepción Legal**: Si hay una orden judicial, mantenemos datos según lo requiera.
