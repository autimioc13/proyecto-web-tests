## ¿Qué Son Cookies?

Las cookies son pequeños archivos de texto que se guardan en tu dispositivo cuando visitas quizlab.com.

```
COOKIE TÍPICA:
├─ Nombre: auth_token
├─ Valor: abc123xyz456
├─ Dominio: quizlab.com
├─ Expiración: 24 horas
└─ Seguridad: HttpOnly, Secure, SameSite
```

### Tipos de Cookies

| Tipo | Duración | Ubicación | Propósito |
|------|----------|-----------|-----------|
| **Session** | Se elimina al cerrar | Servidor | Mantener login |
| **Persistent** | Semanas/meses/años | Dispositivo | Recordar preferencias |
| **First-party** | Varía | quizlab.com | Nuestros servicios |
| **Third-party** | Varía | Google, Stripe, etc. | Socios/análisis |

## Nuestras Cookies

### 1. Cookies Necesarias (OBLIGATORIAS)

Estas cookies son REQUERIDAS para que QuizLab funcione. **No se pueden desactivar.**

| Cookie | Duración | Función | Base Legal |
|--------|----------|---------|-----------|
| `auth_token` | 24 horas | Autenticación/login | Contrato (Art. 6.1.b GDPR) |
| `session_id` | Sesión | Mantener sesión activa | Contrato |
| `csrf_token` | Sesión | Proteger contra ataques | Interés legítimo |
| `analytics_minimal` | Sesión | Contar usuarios para uptime | Interés legítimo |
| `cookie_consent_version` | 2 años | Recordar tu elección de cookies | Consentimiento |
| `lang` | 1 año | Tu idioma preferido | Consentimiento |

**Propósito**: Sin estas cookies, no puedes:
- ✅ Iniciar sesión
- ✅ Completar compras
- ✅ Usar la plataforma

### 2. Cookies Funcionales (RECOMENDADAS)

Mejoran tu experiencia pero **puedes desactivarlas.**

| Cookie | Duración | Función | Desactivar |

| `preferences` | 1 año | Tema (oscuro/claro), tamaño texto | [❌ Rechazar] |
| `last_viewed_test` | 30 días | Qué test viste last | [❌ Rechazar] |
| `playback_speed` | 1 año | Velocidad videos (si tenemos) | [❌ Rechazar] |
| `accessibility_mode` | 1 año | Modos accesibilidad | [❌ Rechazar] |

**Propósito**: Recordar tus gustos personales.

## Cookies por Región

### GDPR (UE/EEE)

```
REQUERIMIENTO:
├─ Opt-in (pedir permiso ANTES de cookies no necesarias)
├─ Banner claro y accesible
├─ Fácil de rechazar como aceptar
├─ Información sobre terceros
└─ Renovación periódica de consentimiento

CUMPLIMIENTO QUIZLAB:
✅ Banner debe aparecer antes de instalar cookies
✅ No instala cookies hasta que aceptes
✅ Botón rechazar igual de prominente que aceptar
✅ Lista completa de cookies en esta política
✅ Cada 90 días pide reconfirmación
```

### CCPA (California)

```
REQUERIMIENTO:
├─ Derecho de saber qué cookies usamos
├─ Derecho de optar por no participar (opt-out)
├─ No discriminar si rechazan cookies
└─ Privacidad de menores < 13 años

CUMPLIMIENTO QUIZLAB:
✅ Lista transparente de cookies
✅ Botón [Optar por No Participar] visible
✅ Mismo servicio aunque rechaces marketing cookies
✅ Bloqueo automático de cookies para < 13
```

### Habeas Data (Colombia)

```
REQUERIMIENTO:
├─ Consentimiento previo y explícito
├─ Información clara sobre cookies
├─ Derecho de acceso a tus datos
├─ Derecho a revocar consentimiento
└─ Uso para fin específico solamente

CUMPLIMIENTO QUIZLAB:
✅ Banner antes de cualquier cookie
✅ Política disponible en español
✅ [Mi Perfil] muestra datos recolectados
✅ [Revocar Consentimiento] en configuración
✅ No usamos para fins diferentes
```

## Cómo Gestionar Tus Cookies

### A Nivel de Navegador

**Chrome:**
1. Abre Configuración
2. Privacidad y Seguridad → Cookies
3. Bloquea terceros o todas

**Safari:**
1. Preferencias → Privacidad
2. "Bloquear todas las cookies"

**Firefox:**
1. Preferencias → Privacidad
2. Cookies → Personalizado

**Edge:**
1. Configuración → Privacidad
2. Elija qué borrar

### A Nivel de Sitio (QuizLab)

**Via Panel:**
```
www.quizlab.com/account/preferences/cookies
└─ Toggle cada tipo de cookie
└─ Guardar inmediatamente
```

### Eliminar Cookies Existentes

**Via Navegador:**
```
Chrome: Configuración → Datos de navegación → Borrar cookies
Safari: Historial → Borrar historial → Cookies
Firefox: Historial → Limpiar historial reciente → Cookies
```

**Via Código (Desarrolladores):**
```javascript
// Eliminar cookie auth_token
document.cookie = "auth_token=; expires=Thu, 01 Jan 1970 00:00:00 UTC; path=/;";
```

### Herramientas para Rechazar Tracking

```
GOOGLE ANALYTICS:
├─ Extensión: https://tools.google.com/dlpage/gaoptout
├─ O via: https://myaccount.google.com/data-and-privacy
└─ Afecta todos los sitios

FACEBOOK PIXEL:
├─ Via: https://www.facebook.com/ads/preferences
├─ O: Configuración → Anuncios
└─ Afecta todos los sitios

RED DE PUBLICIDAD (NAI):
├─ Via: https://optout.networkadvertising.org
├─ Rechaza publicidad dirigida de 100+ compañías
└─ Afecta todos los sitios

DO NOT TRACK (DNT):
├─ Configuración del navegador
├─ Algunos sitios lo respetan
└─ QuizLab respeta DNT (si lo estableces)
```

### Cómo Configurar DNT

**Chrome:**
1. Configuración → Privacidad y seguridad
2. Bajar → "Enviar solicitud 'No rastrear'"

**Safari:**
1. Preferencias → Privacidad
2. Marca "Pedir no rastrear"

**Firefox:**
1. Preferencias → Privacidad
2. "Decir a los sitios que no quiero ser rastreado"

**QuizLab respeta DNT:** Si lo tienes activado, limitamos cookies de marketing.

### Almacenamiento de Consentimiento

```javascript
// Guardar consentimiento
function saveCookieConsent(level) {
  const expires = new Date();
  expires.setDate(expires.getDate() + 90);
  
  document.cookie = `cookie_consent=${level}; 
    expires=${expires.toUTCString()}; 
    path=/; 
    secure; 
    samesite=strict`;
}

// Verificar si aceptó
function hasCookieConsent(type) {
  const consent = getCookie('cookie_consent');
  return consent === 'all' || (consent === 'necessary_only' && type === 'necessary');
}
```
